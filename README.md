# E-commerce-Sales-Data-Analysis-in-Power-BI
Analyzed e-commerce sales data using Power BI by integrating multiple datasets including Orders, Order Details, and Sales Targets.

Project Overview

This project analyzes e-commerce sales data using Microsoft Power BI.
The goal of the project is to understand sales performance, profit trends, and order patterns using data analysis and visualization.

The dashboard helps identify important insights about product categories, sales targets, and regional sales performance.

Dataset

The project uses three datasets:

List of Orders
Order Details
Sales Target

These datasets contain information about orders, product categories, profits, quantities, sales targets, and locations.

Data Preparation

Before creating visualizations, the data was prepared by:

Creating relationships between the tables
Creating calculated columns to categorize and organize the data
Creating measures to calculate key metrics such as order count, profit, and sales performance
Data Visualizations

Several visualizations were created to analyze sales performance:

Clustered Column Chart – compares actual sales with sales targets by category
Donut Chart – shows maximum profit margin by product sub-category
Line Chart – displays monthly sales trends
Scatter Chart – compares profit and quantity across sub-categories
Cards and Multi-row Cards – display total sales amount and sales targets
Matrix – compares sales performance across categories and months
Map – shows geographic sales distribution by city
Treemap – displays sales distribution across product sub-categories
Funnel Chart – analyzes order count distribution by state
Dashboard

An interactive dashboard was created to combine all visualizations and provide a clear overview of sales performance and trends.
